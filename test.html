Oracle to Redshift SQL Transformation Examples

이 문서는 Oracle SQL 구문을 Amazon Redshift에 맞게 변환하는 예제를 제공합니다. 각 예제에는 As-Is(원본) 쿼리와 To-Be(변환 후) 쿼리를 포함하며, 사용 용도, 변환 시 주의할 점, 스크립트 자동화 가능 여부를 명시합니다.

목차

  1. 선언/제어문
  2. 데이터 타입 변환
  3. 날짜/시간 함수 변환
  4. 수학 함수 변환
  5. 문자열 함수 변환
  6. 널 처리 및 CASE
  7. 시퀀스
  8. 예외 처리
  9. 계층 쿼리
  10. 기타 변환 (힌트, 스토리지 옵션 등)

1. 선언/제어문

PL/SQL 익명 블록을 Redshift의 PL/pgSQL로 변환할 때 사용합니다.

Category용도As-Is (Oracle)To-Be (Redshift)스크립트주의
DECLARE 변수 선언 및 초기화 
DECLARE my_var NUMBER;
BEGIN
  my_var := 1;
END;
 
DO $$
DECLARE my_var INTEGER;
BEGIN
  my_var := 1;
END;
$$ LANGUAGE plpgsql;
 불가능 데이터 타입 매핑 수동 검토 필요

  NULL;
END;
 
BEGIN;
  -- SQL 문
END;
 불필요 구문 차이 없음
